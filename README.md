# 🎤 Voice Emotion & Timbre Analyzer

Приложение на Python с GUI с возможностью сохранять файлы для анализа голоса: определяет **эмоцию** говорящего, его **пол** и визуализирует **тембр речи** через спектрограммы и MFCC.

## 🧠 О проекте

Учебный проект для изучения **Machine Learning** в задачах обработки звука. Полный ML-пайплайн:

1. Предобработка аудиоданных
2. Извлечение признаков (MFCC, спектральные характеристики) с помощью `librosa`
3. Обучение собственных моделей классификации на датасете Dusha
4. Интеграция моделей в десктопное приложение с GUI (PySide6)
5. Сохранение истории обработок и результатов локально (SQLite)

## 🛠️ Технологии

| Компонент | Инструмент |
|:----------|:-----------|
| Язык | Python 3.10 |
| GUI | PySide6 |
| Анализ звука | Librosa |
| Машинное обучение | Scikit-learn |
| Визуализация | Matplotlib |
| Управление пакетами | Conda |
| Хранение истории | SQLite |

## 📊 Задачи

- **Классификация эмоций** (радость, грусть, злость, нейтрально)
- **Классификация пола диктора** (мужской, женский)
- **Визуализация тембра** (MFCC heatmap, спектрограмма)

## 📥 Вход

Аудиофайл `.wav`, загружаемый пользователем через GUI.

## 📤 Выход

- `report.txt` — предсказания модели (эмоция, пол)
- `mfcc_heatmap.png` — тепловая карта MFCC
- `spectrum_plot.png` — спектральный портрет голоса
